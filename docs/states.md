# 화면 · 상태 정의

프로토타입에서 다루는 상태와, **구현 여부**를 정리했습니다.

- **구현** — `create` / `respond` / `merge` / `overlays`에서 클릭으로 확인 가능
- **설계** — `components.html` 또는 본 문서에만 정의 (의도 명시)

## 공통 피드백

| 유형 | 용도 | 예시 | 구현 |
|------|------|------|------|
| Toast · success | 완료 확인 | 일정 입력을 완료했어요 | ✅ |
| Toast · warning | 가벼운 검증 | 참석자 이름을 입력해주세요 | ✅ |
| Bottom sheet | 되돌리기 어려운 액션 | 회의 확정 / 요청 발송 | ✅ |
| Inline error | 필드 단위 | 이름을 확인해주세요 | ✅ |
| Empty state | 후보 없음 | 가능한 시간이 없어요 | ✅ |
| Disabled CTA | 선행 조건 미충족 | 시간을 선택해주세요 | ✅ |
